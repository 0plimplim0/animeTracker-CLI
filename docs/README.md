# AnimeTracker-CLI v1.0

## Estado del proyecto

En desarrollo activo. Version 1.0 estable.
## Qué es?
Es un programa CLI para trackear los animes que ves de forma local desde la terminal.

## Cómo se usa?
1. Tener instalado Python.
2. clonar el repo.
    ```
    git clone https://github.com/0plimplim0/animeTracker-CLI
    ```
3. Crear la carpeta **/logs/**.
4. Ejecutar el programa.
    ```
    python ./main.py -h     // Este comando te muestra todas las opciones del programa.
    ```

## Features
- Agregar animes.
- Avanzar episodios.
- Actualizar el estado de los animes agregados.
- Buscar animes por su titulo y ver su informacion.
- Buscar animes por estado asignado.
